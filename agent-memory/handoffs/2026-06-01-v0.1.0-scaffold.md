# Handoff - v0.1.0 scaffold

_This is the current state. Newer handoffs supersede it; relink "latest" in

## State (DONE & verified)

The repo is a Rust crate workshop, docker-first, edge-ready at the core.

- **`crates/drone-dsp`** - `no_std`-friendly DSP core: Hann window, real FFT
  (`microfft`), magnitude spectrum, spectral features. 5 unit tests.
- **`crates/drone-detect`** - `no_std`-friendly heuristic detector
  (band-ratio + dominant-in-band). 3 unit tests.
- **`crates/drone-cli`** - `std` binary `drone` with `synth` and `analyze`.
- **Docker**: `Dockerfile` (host binary), `Dockerfile.dev` (toolchain w/
  rustfmt+clippy), `docker-compose.yml` (`detector` + `dev`),
  `scripts/check.sh`.
- **Memory**: this `agent-memory/` store, indexed by [MEMORY.md](../MEMORY.md).

### Verified 2026-06-01
- Host: all 8 tests pass; synth + analyze discriminate (see
  [insight](../insights/synthetic-signal-discriminates.md)).
- Docker: image builds; `docker compose run --rm detector` synth+analyze works
  through the `./data` mount.
- `docker compose run --rm dev` → **ALL CHECKS PASSED** (fmt, clippy
  `-D warnings`, tests, `--no-default-features` no_std build).
